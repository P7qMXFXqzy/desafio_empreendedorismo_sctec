Este software foi feito para o desafio de IA para DEVs da SCTEC.

A linguagem escolhida foi Python, utilizando a biblioteca PyQT5 para o frontend e salvamento dos dados em json, utilizando uma abordagem simples porém funcional.

O sistema apresenta duas telas, uma com a grid onde apresenta todos os dados no arquivo dados.json, onde os empreendimentos são salvos, onde as linhas são selecionáveis e após 
pressionadas duas vezes sobre uma, abre a tela de edição de dados com os dados já carregados da linha selecionada.

Já na tela de edição de dados, é possível tanto editar, criar, ou deletar um empreendimento. Onde, para criar, nenhum dos campos deve estar vazio. Para editar, basta selecionar uma das 
linhas da grade da tela de listas e já abrirá a edição do empreendimento selecionado. Para deletar um empreendimento, basta selecionar um e apertar no botão escrito "deletar" na tela de 
edição e confirmar na janela que aparecerá.

Os registros no JSON são salvos por IDs numéricos que são automaticamente aumentados conforme os registros são salvos.
